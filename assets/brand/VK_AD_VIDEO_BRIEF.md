## 2. Описание (Lead Magnet) — макс. 16 384 знака

**ALGORA — ИИ-система, которая находит прибыльные товары из Китая для селлеров Wildberries и Ozon**

Пока вы ищете один товар вручную, наш ИИ уже проанализировал сотни и отобрал лучшие.

Каждый день система Algora мониторит тысячи позиций на крупнейших китайских площадках — 1688, Alibaba, Taobao — и выдаёт готовый концентрат: только те товары, которые реально можно продать на российском рынке с хорошей маржой.

Это не подборка ссылок. Это полноценный аналитический отчёт по каждому товару:

— Цена FOB от фабрики (в юанях и рублях)
— Полная себестоимость с учётом доставки, таможни и логистики до склада
— Средняя цена и количество конкурентов на Wildberries
— Расчётная маржа в процентах
— Проверка поставщика: стаж работы, рейтинг, объём продаж
— ИИ-оценка рисков и рекомендация по входу в нишу

Всё, что нужно для принятия решения — в одной карточке.

**Кому это нужно:**

Действующим селлерам WB и Ozon, которые устали от ручного поиска и хотят находить новые ниши быстрее конкурентов.

Начинающим предпринимателям, которые хотят начать торговлю с Китаем, но не знают, с какого товара стартовать и где искать надёжного поставщика.

Опытным импортёрам, которые расширяют ассортимент и хотят принимать решения на основе данных, а не интуиции.

**Как это работает:**

Система обрабатывает объём данных, который один человек не может охватить физически. ИИ-агенты круглосуточно сканируют китайские площадки, отслеживают растущие тренды, сопоставляют с данными российских маркетплейсов — и выдают только те позиции, где есть реальный потенциал прибыли.

Никакой воды. Никакой мотивации. Только цифры, факты и расчёты.

Подписывайтесь на канал @algora_trends — находки публикуются ежедневно.

Корпорация Алгора

## 3. Промпт для генерации видео (AI Video / Motion Design)

### Технические требования VK
| Параметр | Значение |
|----------|----------|
| Формат | MP4 (H.264) |
| Разрешение | 1080x1080 px (квадрат 1:1) — рекомендовано для ленты, мин. 600x600 |

### Промпт для AI-генерации видео

```
Create a 15-second square (1080x1080) motion graphics video for the brand ALGORA — an AI analytics platform for e-commerce sellers who source products from China.

VISUAL STYLE:
- Minimalist, geometric, clean. SaaS/fintech aesthetic, NOT infobusiness.
- Dark theme: deep navy background (#0F172A) with subtle grid pattern overlay.
- Accent colors: Data Violet (#6366F1) for glowing lines, gradients, UI elements; Margin Mint (#10B981) for profit indicators, CTA highlights, success markers.
- Light text (#F1F5F9) for readability on dark background.
- No stock imagery. No robots, brains, gears, lightbulbs, or Chinese characters.
- Think: analytical dashboard, data visualization, modern tech interface.

TYPOGRAPHY:
- Headlines: Manrope SemiBold/Bold, 600-700 weight
- Body text: Inter Regular/Medium, 400-500 weight
- Numbers and data: JetBrains Mono, 400-500 weight
- All text in Russian

SCENE SEQUENCE (15 seconds):

[0-3s] HOOK — Problem Statement
- Dark background with subtle animated dot-grid pattern
- Text animates in (fade + slight upward motion):
  "Ищете товары из Китая вручную?"
  (Manrope Bold, 36-40px, white #F1F5F9)
- Small animated clock icon (line style, 1.5px stroke) spinning fast, symbolizing wasted time
- Subtle violet (#6366F1) glow pulses behind the text

[3-7s] SOLUTION — AI Working
- Smooth transition: the dot-grid transforms into a stylized world map outline (Russia ← China connection)
- Animated data streams (thin violet lines with small glowing dots) flow from China-side to Russia-side
- Abstract product cards (small rounded rectangles) appear and get scanned/filtered by an invisible force
- Some cards fade out (rejected), some get highlighted with green (#10B981) border (selected)
- Small floating numbers appear near selected cards: "Маржа: 67%", "Рейтинг: 8.2" in JetBrains Mono
- Text overlay (bottom): "ALGORA анализирует тысячи товаров" (Inter Medium, 20px)

[7-11s] PROOF — Product Card Preview
- A stylized product analytics card slides in from the right side:
  ┌──────────────────────────┐
  │  Категория: Электроника  │  (Inter, small)
  │                          │
  │  Себестоимость: 1 250 ₽  │  (JetBrains Mono)
  │  Цена WB:      3 800 ₽  │  (JetBrains Mono)
  │  Маржа: 67% ████████░░  │  (green #10B981 bar)
  │  Конкуренция: низкая     │  (Inter)
  │  AI Score: 8.2/10        │  (JetBrains Mono, green)
  └──────────────────────────┘
- Card has dark interface background (#1E293B), rounded corners (12px), subtle violet border glow
- Numbers animate counting up (from 0 to final value)
- Green checkmark appears next to "AI Score" with a satisfying micro-animation

[11-15s] CTA — Call to Action
- Card scales down slightly and moves to the left
- Right side: ALGORA logo (stylized geometric "A") fades in with a subtle glow
- Below logo: "ALGORA" text (Manrope SemiBold, wide letter-spacing)
- Below brand name: "@algora_trends" (Inter Regular, #6366F1 violet)
- CTA button appears at bottom center:
  Rounded rectangle, Margin Mint (#10B981) fill
  Text: "Подписаться" (Inter Medium, white, 18px)
  Subtle pulse animation on the button (scale 1.0 → 1.02 → 1.0, repeating)
- Tagline above CTA: "Находки каждый день. Бесплатно." (Inter Regular, 16px, #F1F5F9)

ANIMATION PRINCIPLES:
- All transitions smooth and elegant (ease-in-out, 300-500ms)
- No harsh cuts — use morphing, fading, sliding
- Subtle parallax depth effect on background grid
- Micro-animations on data elements (gentle floating, pulsing)
- Overall feel: calm confidence, precision, intelligence
- No flashy/aggressive animations — this is a premium analytical tool, not a sale banner

MOOD REFERENCE:
- Linear.app product videos
- Vercel dashboard aesthetics
- Stripe product announcements
- Modern SaaS onboarding animations

DO NOT INCLUDE:
- Human faces or stock photos
- Emojis or cartoon elements
- Chinese characters or flags
- Aggressive sale language ("СКИДКА!", "БЕСПЛАТНО!!!")
- Gradient rainbow colors or neon
- 3D renders or glossy effects
```

Ни один AI-генератор не отрисует кириллический текст безупречно.
Лучший подход — разделить работу:

**Этап 1 — AI генерирует фон (без текста)**

```
Abstract dark tech background for a social media ad, 1080x1080 square.

Deep navy (#0F172A) background with:
- Subtle animated-looking dot-grid pattern (like a data dashboard)
- Faint violet (#6366F1) gradient glow in the upper right area
- Thin geometric lines suggesting data flow / neural connections
- A floating dark card (#1E293B) on the right side with rounded corners and a soft violet glow border — the card is blank/placeholder (no text)
- Small abstract data visualization elements: thin lines, small dots, subtle bar charts

Style: minimalist, modern SaaS, fintech dashboard aesthetic.
Colors: strictly #0F172A, #1E293B, #6366F1, #10B981 (small green accents), #F1F5F9.
No text. No people. No 3D. No stock imagery. Clean, premium, professional.

--ar 1:1 --style raw --v 6.1
```

**Этап 2 — Добавить текст в Canva / Figma**

- CTA-кнопка: "Подписаться" — Inter Medium, белый на зелёном #10B981
