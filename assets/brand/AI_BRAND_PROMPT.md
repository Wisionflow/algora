# Промпт для разработки фирменного стиля ALGORA

Используй этот промпт в ChatGPT-4, Claude или другом AI для генерации бренд-гайда.
Для визуальных материалов (логотип, паттерны) — Midjourney / DALL-E / Ideogram.

---

## Промпт (для текстового AI — Claude / ChatGPT)

```
Ты — бренд-дизайнер и стратег. Разработай полный фирменный стиль для проекта ALGORA.

## О проекте

ALGORA — AI-платформа для селлеров Wildberries и Ozon. Система автоматически мониторит китайские фабрики (1688.com), анализирует тренды и выдаёт готовые товарные находки с расчётом маржи, оценкой конкуренции и рекомендацией поставщика.

## Целевая аудитория

- Селлеры маркетплейсов WB/Ozon (25-45 лет)
- Начинающие предприниматели, которые хотят торговать с Китаем
- Опытные импортёры, расширяющие ассортимент
- География: Россия, СНГ
- Уровень дохода: средний+
- Ценности: эффективность, данные, конкурентное преимущество

## Позиционирование

- "Ваш AI-аналитик китайского рынка"
- Не просто парсер, а интеллектуальная система оценки
- Экономит часы ручного поиска → готовые решения каждый день
- Технологичность + доступность (не сложно, а умно)

## Личность бренда

- Профессиональный, но не корпоративный
- Технологичный, но понятный
- Уверенный, но не агрессивный
- Дружелюбный эксперт, а не холодная система

## Каналы использования

1. Telegram-канал (основной) — посты, карточки товаров
2. VK-сообщество — кросс-постинг
3. Лендинг (будущий) — algora.pro или аналог
4. Карточки товаров (изображения для постов)
5. PDF-отчёты (лид-магниты, еженедельные дайджесты)
6. Аватар/иконка для мессенджеров

## Что нужно разработать

### 1. Цветовая палитра
- 2-3 основных цвета (primary, secondary, accent)
- 2 нейтральных (фон, текст)
- Цвета должны: передавать технологичность + надёжность + рост
- Укажи HEX, RGB, и описание каждого цвета
- Учти контрастность для Telegram (тёмная и светлая тема)

### 2. Типографика
- Заголовочный шрифт (характер, стиль)
- Основной текстовый шрифт
- Моноширинный шрифт (для цифр, цен)
- Только Google Fonts (бесплатные, веб-совместимые)
- Обоснуй выбор каждого шрифта

### 3. Логотип — концепт
- Опиши идею логотипа (символ + текст)
- Варианты: полный, иконка, монохром
- Какие образы использовать (графика, абстракция, буквы)
- Чего избегать
- НЕ рисуй логотип, только опиши концепцию для дизайнера

### 4. Визуальный стиль
- Стиль иллюстраций и графики
- Стиль фотографий (если используются)
- Иконки — стиль, толщина линий
- Паттерны и текстуры
- Примеры do/don't

### 5. Тон голоса (Tone of Voice)
- Как бренд общается с аудиторией
- Примеры фраз: приветствие, CTA, ошибка, успех
- Какие слова использовать / избегать
- Стиль для Telegram vs лендинг vs PDF

### 6. Шаблоны — описание
- Карточка товара (Telegram-пост): что где расположено
- Обложка VK: размеры, элементы
- Герой лендинга: композиция
- PDF-отчёт: макет страницы

### 7. JSON с цветами
Сгенерируй файл palette.json:
{
  "primary": {"hex": "#...", "rgb": "...", "name": "..."},
  "secondary": {"hex": "#...", "rgb": "...", "name": "..."},
  "accent": {"hex": "#...", "rgb": "...", "name": "..."},
  "background": {"hex": "#...", "rgb": "...", "name": "..."},
  "text": {"hex": "#...", "rgb": "...", "name": "..."},
  "success": {"hex": "#...", "rgb": "...", "name": "..."},
  "warning": {"hex": "#...", "rgb": "...", "name": "..."},
  "error": {"hex": "#...", "rgb": "...", "name": "..."}
}

Выдай результат структурированно, с обоснованием каждого решения.
```

## Промпт для Midjourney / DALL-E (логотип)

После получения концепции от текстового AI, используй эти промпты для генерации визуалов:

### Логотип
```
Logo design for "ALGORA" — AI analytics platform for e-commerce sellers.
Minimalist, modern, tech-forward. Symbol: abstract data/trend arrow combined
with letter A. Colors: [вставить цвета из палитры]. Clean vector style,
suitable for app icon and social media avatar. White background.
--style raw --ar 1:1
```

### Паттерн/фон для постов
```
Seamless geometric pattern for tech brand, subtle data visualization motif,
[primary color] on [background color], minimalist, suitable for social media
card background. --tile --ar 1:1
```

### Карточка товара (шаблон)
```
Social media card template mockup, product spotlight design, modern tech style,
[brand colors], clean layout with price tag, rating stars, trend arrow,
margin percentage badge. Professional e-commerce analytics aesthetic.
--ar 9:16
```

## Промпт для Ideogram (логотип с текстом)

Ideogram лучше всего рендерит текст в изображениях:

```
Minimalist logo "ALGORA" with tagline "AI Market Intelligence".
Modern sans-serif typography, abstract upward arrow/data symbol integrated
into letter A. [Primary color] on white. Vector style, clean, professional.
```

## После генерации

1. Выбери лучший вариант логотипа
2. Попроси AI доработать в нужную сторону
3. Финальную версию отрисуй в Figma (вектор)
4. Сохрани все варианты в `assets/brand/logo/`
5. Заполни `BRAND_GUIDE.md` финальными решениями
6. Обнови `palette.json` в `assets/brand/colors/`
