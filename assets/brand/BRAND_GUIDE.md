# ALGORA Brand Guide

## Миссия

ALGORA — AI-аналитик китайского рынка для селлеров маркетплейсов.
Экономим часы ручного поиска, выдаём готовые товарные находки каждый день.

**Ключевая цепочка:** Данные → Решения → Прибыль

**Семантика названия:**
- **Algo** — алгоритм
- **Agora** — рынок (др. греч.)

## 3. Логотип

### Концепция

Геометрическая буква **A**, построенная из:
- Стрелок роста (тренд вверх)
- Сетки данных / графика
- Сегментов (AI-анализ)

**Образ:** AI находит прибыльные направления.
**Форма:** треугольник вверх + цифровая сетка.

**Альтернативная идея:** шестиугольная форма (AI-ячейка) + стрелка вверх.

### Варианты

| Вариант | Файл | Описание |
|---------|------|----------|
| Полный | `logo/logo_primary.svg` | Символ A + текст "ALGORA", шрифт Manrope SemiBold |
| Иконка | `logo/logo_icon.svg` | Только символ A (для Telegram, favicon, приложений) |
| На тёмном фоне | `logo/logo_dark.svg` | Белый/светлый вариант |
| Монохром | `logo/logo_mono.svg` | Чёрный и белый (для печати, документов) |

### Размеры

- Минимальный размер иконки: **32px**
- Минимальный размер полного: **120px**
- Защитная зона: **1/4 высоты** логотипа со всех сторон

### Чего избегать

- Мозги, роботы, шестерёнки
- Китайские иероглифы
- Графики в stock-стиле
- Лампочки, молнии
- Всё, что ассоциируется с инфобизнесом

**Цель:** выглядеть как SaaS-продукт, а не инфобизнес.

## 4. Визуальный стиль

### Графика

- **Стиль:** минимализм, геометрия, линии, сетки, диаграммы, карточки
- **Ассоциации:** дашборд, аналитическая система, финтех

### Иллюстрации

- **Тип:** изометрия, линии, абстрактные формы, data-визуализация
- **Можно показывать:** коробки товаров, графики, нейросеть, логистику

### Фотографии (если используются)

- **Сюжеты:** товары, руки с упаковкой, склад, ноутбук с аналитикой
- **Стиль:** холодный свет, технологичная обработка
- **Нельзя:** стоковые "улыбчивые команды"

### Иконки

- **Стиль:** line icons, толщина 1.5-2px, скругления, минимализм
- **Библиотеки:** Lucide, Phosphor, Tabler

### Паттерны

- Точки данных
- Сетки
- Диагональные линии
- Цифровые волны

### Do / Don't

| DO | DON'T |
|----|-------|
| Много воздуха | Перегруженные карточки |
| Цифры крупно | Кислотные цвета |
| Контраст | Инфобизнес-баннеры |
| Аналитический стиль | Золото и роскошь |

## 5. Тон голоса (Tone of Voice)

### Архетип

**Эксперт-аналитик + партнёр**

Мы — AI-ассистент, который помогает зарабатывать.
Не учитель, не гуру, не продавец курсов.

### Характер речи

- Уверенный
- Конкретный
- Без воды
- Дружелюбный
- Понятный

### Принципы

| Говорим | Не говорим |
|---------|-----------|
| Цифрами | Мотивацией |
| Фактами | "Успешный успех" |
| Выгодой | Сложным техно-языком |

### Примеры фраз

| Ситуация | Пример |
|----------|--------|
| Приветствие | "Наш AI нашёл новую товарную возможность." |
| CTA | "Посмотреть расчёт" / "Забрать подборку" / "Проверить нишу" |
| Успех | "Ниша показывает высокий потенциал прибыли." |
| Ошибка | "Не удалось загрузить данные. Попробуйте позже." |

### Стиль по каналам

| Канал | Стиль |
|-------|-------|
| Telegram | Коротко. Конкретно. Цифры. |
| Лендинг | Более продающий, но рациональный |
| PDF / отчёты | Экспертный. Аналитический. |
