## Методология

Каждая гипотеза оценивается по 7 критериям из манифеста (раздел 4.3).
Шкала: 1-5 (1 = слабо, 5 = отлично).

**Веса критериев (v1.0 — устаревшие):**
- Высокий (x3): Острота боли, Фактоёмкость, Доступность источников
- Средний (x2): Размер аудитории, Платёжеспособность, Связанность с другими сегментами
- Низкий (x1): Конкуренция (инвертировано: 5 = мало конкурентов)

**Максимально возможный балл: 85** (5×3 + 5×3 + 5×3 + 5×2 + 5×2 + 5×2 + 5×1)

## ТОП-3: Детальный анализ

---

### #1: T1 — Фабрики электроники (Китай) → Селлеры маркетплейсов (WB/Ozon) — 70 баллов

**Почему лидер:**

**Острота боли (5/5):**
Селлеры маркетплейсов ежедневно теряют деньги на двух вещах: (1) закупают товар, который не продаётся, и (2) пропускают тренды, которые ловят конкуренты. Один неудачный завоз может стоить 500K-2M рублей. Информация о том, что "этот товар начал взлетать в Китае, но его ещё нет на WB" — буквально стоит денег.

**Фактоёмкость (5/5):**
Идеальный сегмент для ИИ-контента. Нужны чистые факты: какой товар растёт, какая маржа, кто производитель, какая цена FOB, какая конкуренция на маркетплейсе. Никаких эмоций, никакой "красивой картинки". Чистая аналитика = чистая ценность.

**Доступность источников (4/5):**
1688.com, Taobao, Pinduoduo — открытые площадки с ценами и объёмами. Тренды видны через динамику заказов. WB/Ozon — API для анализа конкуренции существуют (MPStats, и др.). Минус: китайские площадки требуют обход антибот-защиты и перевод с китайского.

**Размер аудитории (5/5):**
По разным оценкам, на WB и Ozon активно работают 300K-500K+ селлеров. Даже 1% — это 3000-5000 подписчиков, достаточно для критической массы.

**Платёжеспособность (4/5):**
Селлеры — предприниматели, привыкшие платить за инструменты (MPStats, аналитика). Средний чек инструментов: 3K-15K руб/мес. Но много мелких селлеров с минимальным бюджетом.

**Связанность (4/5):**
Естественные связки: T2 (запчасти), T5 (оборудование), H6 (юристы по ВЭД), F4 (финансирование закупок). Селлерам нужны логисты, таможенные брокеры, юристы — это готовые парные сегменты для Фазы 2.

**Конкуренция (2/5):**
Существуют аналитические сервисы (MPStats, Moneyplace), но они дают сырые данные, а не "контент-концентрат". Telegram-каналы с аналитикой есть, но большинство — ручной труд или перепосты. Полностью автоматизированный ИИ-канал с ежедневными находками — это пока пустая ниша.

### #2: H6 — Юристы по международному праву → Экспортёры/Импортёры — 69 баллов

**Почему на втором месте:**

**Острота боли (5/5):**
Санкционное право меняется еженедельно. Один пропущенный указ ЦБ или новый пакет санкций ЕС может заблокировать поставку на миллионы рублей. Бизнес буквально не успевает отслеживать изменения. Ошибка стоит не просто денег — штрафы, конфискация товара, уголовные риски.

**Фактоёмкость (5/5):**
Чистейший факт-контент: "Вчера ЦБ выпустил указание №XXX — теперь оплата в юанях через банк Y требует документа Z". Никакой воды. Каждый факт = конкретное действие для бизнеса.

**Доступность источников (4/5):**
Официальные источники: Консультант+, OFAC, EU Official Journal, ФТС России, сайты ЦБ. Всё публично, структурировано, обновляется регулярно. ИИ может мониторить и переводить в реальном времени.

**Связанность (5/5)— лучший показатель среди всех гипотез:**
Юридическая информация нужна ВСЕМ кросс-граничным связкам: T1, T2, T4, T5, F1, F2. Это "горизонтальный" сегмент, который естественно соединяется с любым товарным или финансовым потоком. Идеальный фундамент для платформы.

**Слабость:**
Размер аудитории (3/5) — экспортёров/импортёров меньше, чем селлеров маркетплейсов. Но каждый из них — высокочековый клиент.

### #3: T2 — Поставщики автозапчастей (Китай) → СТО и сервисные центры (РФ) — 67 баллов

**Почему в тройке:**

**Острота боли (5/5):**
После ухода западных брендов из РФ образовался острый дефицит оригинальных запчастей. СТО тратят часы на поиск аналогов. Машина стоит на подъёмнике, клиент ждёт, а нужной детали нет. Каждый день простоя = потеря выручки.

**Фактоёмкость (5/5):**
Запчасти — это номера, спецификации, таблицы совместимости. "Деталь OEM #12345 заменяется китайским аналогом #67890, цена FOB $3.2, минимальный заказ 100 шт, завод в Гуанчжоу". Чистая фактография.

**Доступность источников (4/5):**
Китайские площадки (1688, Made-in-China) содержат огромные каталоги запчастей с OEM-номерами. Cross-reference таблицы существуют. ИИ может автоматически строить цепочки совместимости.

**Слабость:**
Связанность (3/5) — менее универсальный сегмент, чем H6. СТО не связаны напрямую с инвесторами или юристами (хотя импортная логистика создаёт связь с H6).

## Рекомендация

### Оптимальный выбор для первой "Единицы": **T1 (Селлеры маркетплейсов)**

**Обоснование:**
1. Максимальный балл по скорингу (70)
2. Самая большая аудитория (300K+ селлеров) — быстрый набор подписчиков
3. Аудитория привыкла потреблять контент в Telegram (десятки каналов про маркетплейсы)
4. Высокая связанность → легко масштабировать на парные сегменты в Фазе 2
5. Формат "ИИ нашёл тренд в Китае для маркетплейса" — вирусный, легко объясняемый

### Стратегический второй ход: **H6 (Юристы/ВЭД)**

**Обоснование:**
H6 имеет максимальную связанность (5/5). Когда селлеры из T1 начнут реально импортировать найденные товары — им понадобится юридическая/таможенная информация. Это создаёт естественный мост между двумя аудиториями и первую "связку" для будущей платформы.

### Путь к платформе:
```
Фаза 1: T1 (селлеры) — набираем аудиторию трендами товаров
    ↓
Фаза 2a: H6 (ВЭД/юристы) — подключаем смежную аудиторию
Фаза 2b: T2 (автозапчасти) — копируем модель на нишевой B2B
    ↓
Фаза 3: Платформа, соединяющая "нашёл товар" + "оформил импорт" + "получил запчасть"
```
