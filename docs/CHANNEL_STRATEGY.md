# ALGORA — Стратегия канала и монетизации

**Обновлено:** 2026-02-23
**Контекст:** Синхронизация с PROJECT_MANIFESTO v2.0 — Growth Agent как основной канал привлечения

---

## 1. Основной канал привлечения: Growth Agent

Growth Agent (AI userbot) — основной и приоритетный канал набора аудитории. Ручное привлечение — архитектурный тупик.

**Как работает:**
- Мониторит 10-30 тематических чатов селлеров
- Даёт экспертные ответы на вопросы по теме (поиск товаров, маржа, Китай)
- Когда уместно — упоминает @algora_trends как полезный ресурс
- Никогда не спамит — только полезные ответы с контекстом

**Ожидаемая конверсия:** 100+ подписчиков за первый месяц работы Agent.

## 2. Контент-канал (@algora_trends)

### Текущий формат (бесплатный)
- 1 пост/день (ротация по дням недели: product, niche_review, beginner_mistake, product_of_week, weekly_top)
- Товарные находки с расчётом маржи
- Работает как лид-магнит для Growth Agent

### Будущий формат: Algora PRO (после 500+ подписчиков)
- Платный канал, 490 руб/мес
- Топ-находки с расширенной аналитикой
- Ранний доступ к трендам
- Формат определяется Product Discovery Agent на основе реальных болей аудитории

## 3. Двухуровневая модель ценности (из Манифеста v2.0)

**Уровень 1 (Продуктовый):** AI-продукт для аудитории, определяемый Product Discovery Agent:
- Контент-канал (текущий @algora_trends)
- AI-бот для индивидуального анализа товаров
- SaaS-дашборд с аналитикой ниш
- Другое — что покажут данные

**Уровень 2 (Платформенный):** Соединение аудиторий (Фаза 3 из Манифеста).
