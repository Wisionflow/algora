## 1. Identity & Philosophy

**Name:** ALGORA
**Etymology:** **ALGO**rithm (Computational Logic) + **AGORA** (Ancient Greek Marketplace & Assembly).
**Core Concept:** "The Digital Agora of Intelligence."

**Philosophy:** Каждый рыночный сегмент имеет неудовлетворённые потребности. Разрыв между экономиками, валютами и уровнем жизни разных стран создаёт возможности с высокой маржинальностью. Algora строит систему AI-агентов, которые идентифицируют эти потребности, создают для аудиторий ценность (контент, инструменты, сервисы) и соединяют аудитории для взаимного решения задач.

Algora продаёт не информацию, а **Решения (Solutions)** и **Связи (Connections)**.

**Mechanism (двухуровневая модель):**

1. **Идентификация потребностей:** AI-агенты мониторят рынок и целевые аудитории, выявляют конкретные боли и неудовлетворённые потребности сегментов.

2. **Два уровня создания ценности:**
   - **Уровень 1 (Продуктовый):** Для каждого сегмента создаётся AI-продукт, решающий выявленную потребность — контент-канал, AI-инструмент, SaaS-сервис, бот. Этот уровень привлекает и удерживает аудиторию, окупает расходы на рост и генерирует доход.
   - **Уровень 2 (Платформенный):** Когда набрана критическая масса аудиторий — соединяем их на платформе. Потребность одной группы решается через другую группу. Это основной центр прибыли.

## 2. Vision & Mission

**Vision:** Международная экосистема взаимосвязанных аудиторий, где каждая группа решает проблемы другой — объединённые на единой платформе через доверие, построенное на ценных AI-продуктах. Кросс-граничные разрывы между экономиками — источник маржинальности.

**Mission:** Создать систему AI-агентов, которые автономно набирают целевые аудитории, выявляют их потребности и создают AI-продукты для монетизации — а затем соединяют эти аудитории на платформе для взаимного решения задач.

**Ключевая ставка:** Главный актив Algora — способность быстро и дёшево набирать целевые аудитории с помощью AI. Если эта способность работает, монетизация и масштабирование решаемы.

## 3. AI-агенты (ядро системы)

Algora строится вокруг трёх специализированных AI-агентов:

### 3.1 Growth Agent (Агент роста) — ПРИОРИТЕТ №1
- **Задача:** Автономно набирать целевую аудиторию без участия человека
- **Как:** Мониторит тематические чаты, даёт экспертные ответы, привлекает подписчиков в канал
- **Стек:** Telethon (userbot) + OpenRouter (LLM) + PostgreSQL
- **Статус:** В разработке (Этап 2)
- **Milestone:** 100 подписчиков за 30 дней = доказательство работоспособности

### 3.2 Market Analyst Agent (Аналитик рынка)
- **Задача:** Мониторить рынок, выявлять сегменты с острыми потребностями, определять критическую массу для каждого сегмента, находить кросс-граничные возможности
- **Как:** Анализирует открытые данные, тренды, конкуренцию, разрывы между экономиками
- **Ритм:** Стратегические решения — еженедельно/ежемесячно
- **Нужен:** До набора аудитории (выбирает направление)
- **Даёт "сырьё"** для Product Discovery Agent

### 3.3 Product Discovery Agent (Агент поиска продуктов)
- **Задача:** Мониторить набранную аудиторию изнутри, выявлять боли, предлагать AI-продукты для создания и продажи
- **Как:** Анализирует разговоры в чатах, вопросы, жалобы, запросы подписчиков
- **Ритм:** Постоянный мониторинг, ежедневно
- **Нужен:** После набора аудитории (200-500+ человек)
- **Результат:** Конкретные предложения AI-продуктов с обоснованием спроса

### Последовательность запуска:
```
Growth Agent (сейчас) → Market Analyst (параллельно) → Product Discovery (после набора аудитории)
```

## 4. Strategic Architecture (3 фазы)

Проект строится последовательно. Каждая фаза — фундамент для следующей.

### Фаза 1: Audience Engine (Единица генерации аудитории)
Создать полностью автоматизированную систему, которая:
- **Набирает аудиторию** через AI Growth Agent (автономно, без ручной работы)
- **Создаёт ценность** для аудитории: контент, AI-инструменты, сервисы
- **Мониторит потребности** аудитории через Product Discovery Agent
- **Монетизирует** через AI-продукты (Уровень 1) — окупает рост и генерирует доход

**Экономика Фазы 1:** Доход от AI-продуктов покрывает расходы на рост аудитории. Прибыль на этом этапе — бонус, не цель.

**Результат фазы:** Работающий Growth Agent + растущая аудитория + хотя бы один монетизируемый AI-продукт для этой аудитории.

### Фаза 2: Replication (Масштабирование)
Копировать систему на другие целевые сегменты:
- Каждый новый сегмент выбирается Market Analyst Agent на основе данных
- Сегменты выбираются так, чтобы потребности были связаны с уже набранными аудиториями
- Для каждого сегмента Product Discovery Agent находит и создаёт AI-продукты
- Каждое направление = отдельный контейнер + TG-аккаунт + промпт

**Результат фазы:** Несколько целевых аудиторий с пересекающимися потребностями, каждая монетизируется через AI-продукты.

### Фаза 3: Platform (Соединение)
После набора критической массы — Уровень 2:
- Соединяем аудитории на платформе, где потребность одной группы решается через другую
- Формат платформы определяется на основе реальных данных о потребностях
- Монетизация через транзакции, подписки, premium-доступ

**Результат фазы:** Работающая платформа с доходом. Основной центр прибыли.

## 5. Выбор сегментов

### 5.1 Критерии выбора сегмента

При выборе сегмента оцениваем по 8 критериям:

| Критерий | Описание | Вес |
|----------|----------|-----|
| **Острота боли** | Насколько критична проблема? Теряет ли аудитория деньги/время без решения? | Высокий |
| **Автоматизируемость ценности** | Можно ли с помощью AI создать продукт (контент, инструмент, сервис), решающий потребность масштабируемо? | Высокий |
| **Доступность источников** | Есть ли открытые данные, которые AI может парсить и анализировать? | Высокий |
| **AI-продуктовый потенциал** | Есть ли у аудитории повторяющиеся задачи, которые AI может автоматизировать и за которые будут платить? | Высокий |
| **Размер аудитории** | Достаточно ли людей для критической массы? | Средний |
| **Платёжеспособность** | Готова ли аудитория платить? | Средний |
| **Связанность с другими сегментами** | Есть ли естественный "парный" сегмент для Уровня 2? | Средний |
| **Конкуренция** | Насколько насыщен рынок? | Низкий* |

*Низкий вес конкуренции, потому что если боль острая и решение качественное — конкуренция преодолима.

> **Примечание:** Критерии обновлены (v2.0). Скоринг 30 гипотез (SEGMENT_SCORING.md) будет пересчитан после валидации Growth Agent на первом сегменте.

### 5.2 Географический фокус

**Стартовая площадка:** Россия (первый сегмент T1).

**Стратегическое направление:** Международный рынок. Кросс-граничные разрывы между уровнем жизни, экономиками и валютами создают наибольшую маржинальность. Конкретные направления определяет Market Analyst Agent на основе данных.

Географии с потенциалом:

- **Турция** — логистический хаб между Европой и Азией
- **Европа** — высокая покупательная способность
