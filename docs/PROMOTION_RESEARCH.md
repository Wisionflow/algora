## 1. Состояние рынка

Ниша «товары из Китая для селлеров WB/Ozon» — конкурентная, но не перенасыщенная.

По данным TGStat, в тегах WB/Ozon — 60+ каналов и 20+ чатов для селлеров.

### Основные конкуренты

| Канал | Формат | Подписчики |
|-------|--------|------------|
| B2B-China (товары оптом из Китая) | Закупки 1688/Alibaba | ~7 400 |
| Поставлено! (логистика WB/Ozon) | Логистика селлеров | 5 000+ |
| Продажи на маркетплейсах (SellerFox) | Аналитика товаров | 10 000+ |
| Express-today (импорт из Китая) | Закупки, доставка | 5 000+ |
| Семья на Маркетплейсах | Кейсы + обучение | 3 000+ |

**Вывод:** Каналы в нише растут медленно (1-3K/мес для крупных). Потолок высокий — аудитория селлеров в России 500K+ человек. Каналов с аналитикой маржи по конкретным товарам почти нет. Наш формат уникален.

## 2. Бесплатные методы продвижения

### A. VC.ru — статьи (САМЫЙ ЭФФЕКТИВНЫЙ)
- Раздел «Маркетплейсы» очень активен
- Кейс: +1 500 подписчиков за месяц, ~90₽ за подписчика (фактически бесплатно)
- Конверсия: 5-12% читателей подписываются на канал
- Что писать: Разборы товаров, расчёты маржи, «5 товаров из Китая с маржой 60%+»
- Статьи индексируются в Яндексе — дают трафик месяцами

### B. Взаимопиар с каналами
- 5-10 каналов схожей тематики (НЕ прямых конкурентов)
- Каналы о логистике из Китая, таможне, ВЭД
- Нужно хотя бы 100-200 подписчиков для старта

### C. Комментинг в чатах селлеров
- Десятки чатов селлеров WB/Ozon (500-10 000 участников)
- Экспертные ответы + ссылка на канал
- Не спамить

### D. Яндекс.Дзен
- Дублёр контента с VC.ru
- Стоимость дочитывания от 3₽, перехода ~7.5₽
- Кейс: 26 000 переходов за 3 недели
- Нет API — только ручной постинг

### E. ВКонтакте
- Группы селлеров активны
- Параллельная группа с перенаправлением в TG
- Есть официальный API — можно автоматизировать
- VK Ads (подробнее в разделе платной рекламы)

### F. Одноклассники
- Пропускаем. Аудитория селлеров отсутствует.

### G. YouTube Shorts / TikTok / VK Clips
- Короткие видео с разборами товаров (30-60 сек)
- Формат: «Товар из Китая за 200₽ → продаём за 1500₽ на WB. Маржа 70%»
- Потенциально вирусный, но требует видеопроизводства

## 3. Платная реклама

### A. Посевы в каналах — ЛУЧШЕЕ соотношение цена/результат
- Цена: 15-50₽ за подписчика
- Порог входа: от 3 000₽
- Где искать: telega.in, adsell.io, TGStat

### B. VK Ads — ДЕШЁВЫЙ подписчик
- Цена: 25-35₽ за подписчика
- Таргетинг: группы селлеров WB/Ozon
- Лайфхак: 100 объявлений по 30К аудитории вместо одного на 3М

### C. Telegram Ads
- Цена: 60-90₽ в товарном бизнесе, до 210₽ для B2B
- Мин. вход: от 500€ через реселлеров
- Подходит при 1000+ подписчиков

### D. Яндекс.Директ
- НЕ рекомендуется. 350-400₽ за подписчика, плохая конверсия.

## 4. Контент-стратегия для роста

### Что привлекает подписчиков:
1. Конкретные цифры — «маржа 73%, вход 11К₽»
2. Списки-подборки — «Топ-5 товаров с маржой 60%+» (вирусный формат)
3. Ошибки и провалы — «Потерял 150К на этом товаре»
4. Калькуляторы и инструменты — Telegram-бот с расчётом маржи

### Лид-магниты:
- «База 100 поставщиков с 1688 с проверенными ценами» — PDF
- Калькулятор unit-экономики — Telegram-бот
- Чек-лист «Первая закупка из Китая» — 20 шагов

### Вирусный контент:
- Сравнения: «Цена на 1688 vs цена на WB» (со скриншотами)
- «Сколько зарабатывает селлер на [товар]»

## 5. Собственный сайт

Сейчас НЕ нужен. На этапе 0-1000 подписчиков не окупится.

Альтернативы: VC.ru, Яндекс.Дзен, Notion/Tilda (для лид-магнита).

Нужен при 5000+ подписчиков для SEO-монетизации.

## 6. План действий для @algora_trends

### Этап 1: 0₽ бюджет → 200 подписчиков за 1 мес
1. 2 статьи на VC.ru (раздел «Маркетплейсы»)
2. Вступить в 10 чатов селлеров, экспертные ответы
3. Лид-магнит: PDF «20 товаров из Китая с расчётом маржи»
4. Дублировать контент в Яндекс.Дзен
5. Переименовать канал: «ALGORA | Товары из Китая для WB и Ozon»

### Этап 2: 5 000₽/мес → 500 подписчиков за 3 мес
- VK Ads: 3 000₽/мес → ~100 подписчиков
- Посев в 1-2 каналах: 2 000₽/мес → ~50-100 подписчиков
- Взаимопиар: 2-3 обмена/мес
