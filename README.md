# ALGORA

AI-аналитик китайского рынка для селлеров маркетплейсов (WB/Ozon).

Автоматически парсит товары с 1688.com, анализирует маржу и конкуренцию,
генерирует посты с AI-инсайтами и публикует в Telegram-канал.

## Канал
[@algora_trends](https://t.me/algora_trends)

## Быстрый старт

1. Скопируй `.env.example` в `.env` и заполни ключи
2. `pip install -r requirements.txt`
3. `python -X utf8 -m scripts.run_pipeline --source demo --dry-run`
